# GeoCheckr

A geography exploration game built with React Native (Expo).

## Features

- 📱 QR Code scanning
- 🗺️ Interactive maps (OpenStreetMap)
- 🎙️ Voice input support
- 🌍 200+ locations worldwide
- 👥 2-8 players
- 📊 Score tracking

## Tech Stack

- React Native / Expo
- TypeScript
- OpenStreetMap (Leaflet.js)
- Firebase
- Web Speech API

## Setup

```bash
npm install
npx expo start
```
