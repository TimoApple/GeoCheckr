# GeoCheckr — Vector Ressourcen für Map-Backgrounds

## Kostenlose SVG Quellen

### Topographische / Kontur-Linien
- **Freepik** — "topographic map pattern SVG" (kostenlos mit Attribution)
- **SVGBackgrounds.com** — Generatoren für abstrakte Linien-Muster
- **HeroPatterns.com** — Wiederholbare SVG-Patterns (CC0)

### Karten / Map Icons
- **SVGRepo.com/vectors/map** — Große Auswahl, CC0 / MIT Lizenzen
- **Tabler Icons** — 4200+ Icons, MIT Lizenz (https://tabler-icons.io)
  - `map-2`, `compass`, `route`, `map-pin`, `world`, `navigation`
- **Phosphor Icons** — Flexibel, MIT (https://phosphoricons.com)
  - `map-trifold`, `globe`, `compass`, `path`
- **Lucide Icons** — 1400+ Icons, ISC Lizenz (https://lucide.dev)
  - `map`, `globe`, `compass`, `map-pin`, `navigation`

### Weltkarten / Kontinent-Silhouetten
- **Natural Earth Data** — Public Domain Vektorkarten
- **SimpleMaps.com** — Free SVG Weltkarte
- **Amcharts** — Free SVG Maps (https://www.amcharts.com/free-svg-maps/)

### Google Maps Stil Elemente
- **Mapbox Maki Icons** — Open Source Map Icons (https://github.com/mapbox/maki)
- **Maps Icons Collection** — Google Maps Style (https://mapicons.mapsmarker.com)

## Empfohlene Kombination für GeoCheckr

1. **Hintergrund:** bg-contours.svg (bereit) + bg-grid.svg (bereit)
2. **Icons:** Tabler Icons Set laden (nur die Map-relevanten)
3. **Weltkarte:** Amcharts SVG World Map für Tutorial-Grafik
4. **Muster:** HeroPatterns für subtile Textur-Overlays

## Icon-Download Empfehlung
Nur 20-30 Icons nötig:
- map-pin, compass, globe, route, navigation
- search, camera, qr-code, timer, trophy
- play, pause, skip, check, close, arrow-right, arrow-left
- users, settings, info

→ Tabler Icons reichen komplett, MIT Lizenz = kein Problem
