## Google Play Store

### Titel
GeoCheckr — Geography Game

### Kurzbeschreibung
Geografie-Party-Spiel mit QR-Codes. Finde den Ort auf der Welt!

### Beschreibung
GeoCheckr ist ein innovatives Geografie-Party-Spiel, das digitale und physische Elemente kombiniert.

**Features:**
✓ QR-Code Scanner für Spielkarten
✓ 200 Locations auf allen Kontinenten
✓ 2-8 Spieler möglich
✓ Distanzberechnung mit Haversine-Formel
✓ Spracheingabe für schnelle Antworten
✓ Punktesystem mit Ziel-Score
✓ Tutorial für neue Spieler
✓ Dunkles Theme

**So funktioniert's:**
1. Jeder Spieler scannt seine Karten
2. QR-Code wird gescannt
3. Street-View-Bild erscheint (30s Timer)
4. Spieler nennt nächste Stadt
5. App berechnet Distanz und vergibt Punkte
6. Erster Spieler mit Ziel-Score gewinnt!

**Perfekt für:**
- Familienabende mit Kindern ab 10
- Partys und Spieleabende
- Schulunterricht (Geografie)
- GeoGuessr-Community
- Reise-Enthusiasten

**Technologie:**
- React Native (Expo)
- OpenStreetMap
- Firebase Backend
- Web Speech API
