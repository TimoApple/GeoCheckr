## Vollständige Beschreibung
🌍 **GeoCheckr** — Das ultimative Geografie-Party-Spiel für 2-8 Spieler!

**So funktioniert's:**
1. Scanne einen QR-Code auf deiner Location-Karte
2. Sieh dir das 360° Panorama an
3. Errate die Stadt per Spracheingabe oder Text
4. Punkte für Nähe — wer die meisten Punkte hat, gewinnt!

**Features:**
🎮 2-8 Spieler auf einem Gerät
🌐 10 kuratierte 360° Panorama-Locations (mehr kommen!)
🎤 Sprachsteuerung — nenne die Stadt einfach per Mikrofon
⏱️ Timer mit verschiedenen Schwierigkeitsstufen
🏆 Runden-System mit Spotify-Style Zusammenfassung
🎨 Wunderschönes Dark-Theme Design

**Perfekt für:**
🏠 Spieleabende mit Freunden
🍻 Partys und Events
📚 Geografie-Lernen
🏫 Schulunterricht

**Kein Abo, keine versteckten Kosten!**
