# User Guide — GeoCheckr App

## WILLKOMMEN

GeoCheckr ist ein Geografie-Party-Spiel für 2-8 Spieler. Scanne QR-Codes, betrachte Bilder und errate Orte weltweit!

## SPIEL STARTEN

1. **App öffnen**
2. **"Neues Spiel"** tippen
3. **Spieler hinzufügen** (2-8 Spieler)
4. **Schwierigkeit wählen:**
   - 😊 Leicht: Berühmte Wahrzeichen
   - 🤔 Mittel: Stadtgebiete
   - 🔥 Schwer: Versteckte Orte
5. **Ziel-Score wählen** (5/10/15/20)
6. **"Spiel starten"** tippen

## SPIELABLAUF

### Phase 1: QR-Code scannen
- Scanne den QR-Code auf der Location-Karte
- Oder tippe "QR-Code scannen" zum Testen

### Phase 2: Bild betrachten (30 Sekunden)
- Ein Bild eines Ortes erscheint
- Du hast 30 Sekunden Zeit
- In den letzten 5 Sekunden gibt es Countdown

### Phase 3: Ortsnennung
- Tippe auf die Stadt, die am nächsten zum gezeigten Ort liegt
- Oder nutze Spracheingabe (Mikrofon-Symbol)

### Phase 4: Ergebnis
- Die App berechnet die Distanz
- Punkte werden vergeben:
  - < 100km: 3 Punkte
  - < 500km: 2 Punkte
  - < 2000km: 1 Punkt
  - > 2000km: 0 Punkte

### Phase 5: Nächster Spieler
- Der nächste Spieler ist dran
- Das Spiel geht weiter bis jemand den Ziel-Score erreicht

## SCHWIERIGKEITSGRADE

### 😊 Leicht (P1)
- Berühmte Wahrzeichen (Times Square, Eiffelturm)
- Klare Straßenschilder
- Eindeutige Architektur

### 🤔 Mittel (P2)
- Stadtgebiet, aber nicht sofort erkennbar
- Architektur-Stil als Hinweis
- Vertraute, aber nicht berühmte Orte

### 🔥 Schwer (P3/P4)
- Vororte oder ländliche Gebiete
- Nur Vegetation, Klima, Boden als Hinweis
- Könnte überall sein

## TIPPS

- **Achte auf Details:** Straßenschilder, Sprache, Architektur
- **Klima erkennen:** Palmen = warm, Schnee = kalt
- **Himmelsrichtung:** Sonnenstand kann Hinweise geben
- **Fahrzeuge:** Links/Rechts-Verkehr
- **Beschriftungen:** Welche Sprache/Schrift?

## TROUBLESHOOTING

### QR-Scanner funktioniert nicht
- Kamera-Berechtigung prüfen
- Lichtverhältnisse verbessern
- QR-Code gerade halten

### Bild lädt nicht
- Internet-Verbindung prüfen
- Neu laden versuchen
- Alternativ: Texteingabe nutzen

### Spracheingabe funktioniert nicht
- Mikrofon-Berechtigung prüfen
- Chrome oder Safari nutzen
- Alternativ: Texteingabe nutzen

### App stürzt ab
- App neu starten
- Cache leeren
- Neu installieren
