# GeoCheckr — Projekt Summary

## ÜBERBLICK
GeoCheckr ist ein Geografie-Party-Spiel, das digitale und physische Elemente kombiniert. Spieler scannen QR-Codes auf Spielkarten, betrachten Street-View-Bilder und erraten Orte weltweit.
