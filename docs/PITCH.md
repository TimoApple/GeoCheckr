# GeoCheckr — Pitch Material

## ONE-LINER
Ein Geografie-Party-Spiel, das die Spannung von GeoGuessr auf den Tisch bringt.

## PROBLEM
- GeoGuessr: Nur digital, kein soziales Erlebnis am Tisch
- Brettspiele: Keine interaktive Geografie-Erfahrung
- Party-Spiele: Meistens Karten/Quiz, keine Karten-Mechanik

## LÖSUNG
- QR-Code scan → Street View Bild → Ort erraten
- App + Physisch = Hybrid-Erlebnis
- 2-8 Spieler, ab 10 Jahren, 20-45 Minuten

## MARKT
- GeoGuessr: Millionen Nutzer, KEIN Brettspiel
- Hitster: Zeigt dass QR-Mechanik funktioniert
- Brettspiel-Markt: $13 Mrd, 10% Wachstum
- Lücke: Kein offline GeoGuessr-Brettspiel

## PREISSTRUKTUR
- Basis-Spiel: 19.99 EUR (200 Karten + App-Code)
- Erweiterungen: 9.99 EUR (100-200 Karten)
- Produktion: 4.60 EUR pro Spiel
- Marge: 15.39 EUR (vor Verlag/Vertrieb)

## TECHNIK
- React Native (Expo)
- OpenStreetMap (kostenlos)
- Mapillary (CC-BY-SA, kostenlos)
- Firebase (kostenlos bis 10k Nutzer)
- Web Speech API (kostenlos)

## ZIELGRUPPE
- Familien mit Kindern 10-16
- GeoGuessr-Community (Millionen)
- Party-Spieler
